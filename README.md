# 🛒 Shopping Cart Web App

A simple and interactive **Shopping Cart application** built using **HTML, CSS, and JavaScript**.
This project demonstrates core front-end development concepts such as **DOM manipulation, dynamic UI updates, and cart state management**.

---

## 🚀 Features

* Add products to cart
* Remove products from cart
* Increase / decrease product quantity
* Dynamic price calculation
* Real-time cart updates
* Clean and responsive UI

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Logic and interactivity
* **Git & GitHub** – Version control

---

Shopping-Cart

## ⚙️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/ManavBhatt123/Shopping-Cart.git
```

2. Navigate to the project folder

```
cd Shopping-Cart
```

3. Open **index.html** in your browser.
